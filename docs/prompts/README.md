# Prompts

Os prompts de sistema atualmente em uso por cada agente estão documentados em
[PROMPTS.md](../../PROMPTS.md), na raiz do repositório — mantidos junto aos outros
documentos "vivos" do projeto (README, ROADMAP, CHANGELOG) para ficarem visíveis de
imediato.

Esta pasta é reservada para material de apoio mais extenso que não cabe em
`PROMPTS.md`: variações testadas de um prompt, notas de tuning, exemplos de
respostas de IA usados para validar o parsing de um agente, etc.

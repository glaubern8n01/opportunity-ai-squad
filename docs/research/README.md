# Research

Pasta reservada para pesquisas profundas de mercado/produto que não se encaixam como
uma linha da tabela `analyses` — por exemplo, notas de pesquisa manual, comparativos
entre categorias inteiras, ou o resultado bruto de uma investigação do `ResearchAgent`
(`src/opportunity_squad/agents/research_agent.py`) quando vale a pena preservar em
Markdown além do que já fica salvo em `analyses.summary`.

Convenção sugerida de nome: `<categoria-ou-produto>-<data>.md`.
